Win32CORE - Win32 CORE functions

DESCRIPTION

This library provides the functions marked as [CORE] in the Win32 manpage. See that document for usage information. In cygwin, as of 5.8.6 it is no longer necessary to use this module; the functions should be available even without use Win32CORE; or -MWin32CORE.
